# 招标资质快速识别与提取

## 技能描述

从招标文件中快速识别和提取资质要求，分级为"硬性门槛"和"评审加分项"，并生成待办清单指导后续准备。是投标前分析的子模块，建议在调用"投标项目全流程管理"后立即执行。

---

## 资质分类体系

### A类 — 硬性门槛（不满足则废标）

| 类型 | 典型条款 | 识别关键词 |
|------|---------|---------|
| 营业执照类 | 具有独立法人资格，持有效营业执照 | "独立法人资格"、"有效营业执照"、"在有效期内的" |
| 行业资质类 | 具有相关行业主管部门颁发的资质证书 | "具备XXX资质"、"持有XXX证书"、"须取得XXX认证" |
| 业绩要求类 | 近X年完成过X个类似项目 | "近X年"、"承担过"、"完成过"、"类似项目" |
| 财务要求类 | 注册资本/营业收入/资产负债率要求 | "注册资本不低于"、"近X年营业收入"、"资产负债率" |
| 人员资质类 | 项目负责人须持有XXX证书 | "须具有XXX资质"、"持有XXX证书"、"具备XXX职称" |
| 授权要求类 | 原厂授权/代理资质/委托函 | "原厂授权"、"代理商资质"、"制造厂商授权" |

### B类 — 评审加分项（有则加分，无则不废）

| 类型 | 典型条款 | 识别关键词 |
|------|---------|---------|
| 认证加分 | 通过ISOXXXX认证、CMMI等级 | "通过ISO"、"CMMI"、"具备XXX认证优先" |
| 案例加分 | 具有同类行业成功案例 | "具有XXX行业经验者优先"、"提供案例加分" |
| 荣誉加分 | 获得行业奖项、政府资质 | "获得XXX荣誉"、"行业领军企业"、"瞪羚企业" |
| 团队加分 | 高级工程师数量、博士比例 | "高级工程师"、"注册XXX师"、"专业人员" |

### C类 — 格式要求（不影响资质有效性）

| 类型 | 典型条款 |
|------|---------|
| 盖章要求 | 加盖公章/骑缝章/鲜章 |
| 密封要求 | 正副本分开密封/包封要求 |
| 签字要求 | 法定代表人/授权人签字 |
| 装订要求 | 左侧装订/胶装/不分册 |

## 快速识别流程

### 第一步：定位资质章节

招标文件通常包含以下资质章节：

| 位置 | 章节名称 |
|------|---------|
| 招标文件正文 | "投标人资格要求"、"投标人资质要求" |
| 招标文件正文 | "资格后审"、"资格审查" |
| 招标附件 | "资格证明材料清单"、"资质要求表" |
| 商务条款 | "投标文件组成"、"证明文件" |

**快速定位法**：直接问 LLM "这份招标文件中投标人需要满足哪些资质要求？分别列出硬性门槛和加分项。"

### 第二步：逐条提取

对每条资质要求提取以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| 编号 | 条款序号 | 1、2.1 |
| 条款原文 | 原文描述 | "具有有效的ISO9001质量管理体系认证" |
| 类别 | A/B/C 类 | A类 |
| 类型 | 营业执照/行业资质/业绩/财务/人员/授权 | 行业资质 |
| 具体要求 | 量化或定性描述 | "有效期内的ISO9001认证" |
| 证明材料 | 需要提供什么文件 | ISO9001证书复印件 |
| 我方现状 | 是否满足/满足程度 | 已取得，有效期至2027年 |
| 状态 | ✅满足/⚠️待补充/❌不满足 | ✅满足 |

### 第三步：生成准备清单

基于提取结果，生成待办清单：

```markdown
## 资质准备清单

### A类 — 硬性门槛（必须全部满足）

| # | 条款 | 需要材料 | 我方状态 | 负责人 | 完成时间 |
|---|------|---------|---------|-------|---------|
| 1 | 营业执照 | 营业执照副本（加盖公章） | ✅已备 | 商务 | D-3 |
| 2 | 软件企业认定证书 | 证书复印件 | ✅已备 | 商务 | D-3 |
| 3 | 近3年类似项目业绩≥3个 | 合同关键页+验收报告 | ⚠️待补充1个 | 销售 | D-5 |
| ... | ... | ... | ... | ... | ... |

### B类 — 评审加分项（如有则提供）

| # | 条款 | 需要材料 | 我方状态 | 负责人 | 完成时间 |
|---|------|---------|---------|-------|---------|
| 1 | ISO27001认证 | 证书复印件 | ✅已备 | 商务 | D-3 |
| 2 | 石油石化行业案例 | 项目介绍 | ⚠️待整理 | 技术 | D-5 |

### C类 — 格式要求

| # | 要求 | 注意事项 | 负责人 |
|---|------|---------|-------|
| 1 | 加盖公章 | 公章在有效期内 | 商务 |
| 2 | 左侧胶装 | 不接受订书钉 | 商务 |
```

## 常见条款的快速判断规则

### 业绩要求

| 条款表述 | 判断规则 |
|---------|---------|
| "近3年承担过X个类似项目" | 合同签订时间在3年内 + 项目内容相似度≥70% |
| "每个合同金额不低于X万元" | 合同金额达标 + 有验收报告 |
| "提供项目验收合格证明" | 必须有甲方盖章的验收单 |
| "联合体业绩可累加" | 联合体投标才适用 |

### 人员资质

| 条款表述 | 判断规则 |
|---------|---------|
| "项目负责人须具有XXX证书" | 证书在有效期内 + 项目负责人近6个月社保证明 |
| "高级工程师≥3人" | 职称证 + 社保证明 |
| "PMP持证者优先" | 加分项，不是门槛 |

### 财务要求

| 条款表述 | 判断规则 |
|---------|---------|
| "注册资本不低于X万元" | 以营业执照注册资本为准 |
| "近2年连续盈利" | 看审计报告资产负债表 |
| "资产负债率≤85%" | 看审计报告，计算公式：负债总额/资产总额 |

---

## 注意事项

1. **区分"实质性要求"和"格式要求"**：C类格式要求不是资质门槛，不要混淆
2. **区分"近X年"和"有效期至X年"**：时间边界要抠准
3. **联合体投标时**：各方资质要求可能不同，分别列明
4. **业绩合同要看关键页**：招标方通常要求提供合同关键页（金额、标的、签字盖章）
5. **人员资质需要社保证明**：仅证书不够，还需要证明是本公司正式员工
