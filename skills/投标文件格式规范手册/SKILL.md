# 投标文件格式规范手册

## 技能描述

投标文件的格式规范汇总，包括字体字号、行距、页边距、页眉页脚、目录、盖章要求等。适用于技术应答文件、商务投标文件、价格标书的格式自检和批量修正。

---

## 通用格式参数（中文投标文件基准）

### 字体与字号

| 元素 | 字体 | 字型 | 字号 | 说明 |
|------|------|------|------|------|
| 正文 | 宋体 | 常规 | 小四（12pt） | 默认正文 |
| 正文（可选） | 仿宋 | 常规 | 三号（16pt） | 部分招标文件要求仿宋 |
| 一级标题 | 黑体 | 加粗 | 三号（16pt） | 或小二（14pt） |
| 二级标题 | 楷体 | 加粗 | 三号（16pt） | 或四号（14pt） |
| 三级标题 | 宋体 | 加粗 | 小三（15pt） | 或四号（14pt） |
| 四级标题 | 宋体 | 常规 | 小四（12pt） | 同正文 |
| 页眉页脚 | 宋体 | 常规 | 五号（10.5pt） | 通常灰色或浅色 |
| 目录标题 | 黑体 | 加粗 | 二号（18pt） | 居中 |
| 表格正文 | 宋体 | 常规 | 小四（12pt） | 居中或左对齐 |
| 表格标题 | 宋体 | 加粗 | 小四（12pt） | 居中 |

### 行距与段落

| 参数 | 值 | 说明 |
|------|------|------|
| 正文行距 | 1.5倍 | 标准中文投标文件 |
| 标题行距 | 1.5倍 | 与正文一致 |
| 段前间距 | 0磅 | 标题可适当加大 |
| 段后间距 | 6-12磅 | 正文段落间 |
| 左缩进 | 0字符 | 标准 |
| 首行缩进 | 2字符 | 仅正文段落 |

### 页边距

| 位置 | 尺寸 | 说明 |
|------|------|------|
| 上边距 | 2.54cm | 约1英寸 |
| 下边距 | 2.54cm | 约1英寸 |
| 左边距 | 2.54cm 或 3.17cm | 根据招标文件要求 |
| 右边距 | 2.54cm 或 3.17cm | 根据招标文件要求 |
| 装订线侧 | 通常左侧 | 留足装订空间 |

**常见组合**：
- 宽边距（3.17cm）：技术文件、内容较多时
- 窄边距（2.54cm）：商务文件、格式紧凑时
- **以招标文件要求为准**，无要求时用 3.17cm

## 页面设置

### 纸张与方向

| 参数 | 值 |
|------|------|
| 纸张 | A4（210mm × 297mm） |
| 方向 | 纵向（默认）；图表较多的章节可申请横向 |
| 每页行数 | 自动（由行距和字号决定） |
| 字符间距 | 标准（默认值） |

### 页眉页脚

| 元素 | 内容 | 格式要求 |
|------|------|---------|
| 页眉左侧 | 项目名称 | 宋体五号，居左 |
| 页眉右侧 | 公司名称 | 宋体五号，居右 |
| 页脚左侧 | 章节名称 | 宋体五号 |
| 页脚右侧 | 页码 | 宋体五号，第X页 共Y页 |
| 页眉线 | 1磅黑色单线 | 或无要求 |
| 页脚线 | 可选 | 通常不加 |

## 标题层级规范

### 中文标题常见格式

| 层级 | 编号形式 | 示例 |
|------|---------|------|
| 一级 | 第X章 / 一、 | 第1章 项目概述 / 一、项目背景 |
| 二级 | X.X /（一） | 1.1 项目背景 / （一）政策背景 |
| 三级 | X.X.X / 1. | 1.1.1 政策背景 / 1. 政策背景 |
| 四级 | （1）/ ① | （1）/ ① |
| 正文 | 首行缩进2字符 | 标准段落格式 |

**禁止混用编号体系**：整篇文件使用同一种编号体系。

## 表格规范

| 元素 | 要求 |
|------|------|
| 边框 | 内外框线完整（"表格网格"样式） |
| 表头 | 加粗 + 底纹（浅灰或蓝色）+ 居中 |
| 表内文字 | 宋体小四，居中或靠左对齐 |
| 续表 | 在上一页底部注明"（续）" |
| 表格标题 | 在表格上方，宋体加粗小四 |
| 表格宽度 | 不超过页面宽度，超宽时横向放置 |

---

## 图表规范

| 元素 | 要求 |
|------|------|
| 图题 | 在图下方居中，宋体五号加粗 |
| 表题 | 在表上方居中，宋体五号加粗 |
| 图表编号 | 图1-1 / 表2-3（章序号-图序号） |
| 分辨率 | 印刷要求300dpi以上 |
| 架构图 | 使用专业绘图软件，禁止截图 |
| 截图 | 仅当示意图时使用，需注明来源 |

---

## 目录规范

| 元素 | 要求 |
|------|------|
| 目录标题 | "目 录"（中间空一格），黑体二号居中 |
| 一级标题 | 宋体小三，左对齐 |
| 二级标题 | 宋体小四，左对齐，右端加页码 |
| 页码 | 与正文页码连续，不单独编起始页 |
| 更新方式 | 完成后用Word自动更新目录 |

---

## 盖章与签字

### 盖章要求

| 位置 | 盖章要求 |
|------|---------|
| 封面 | 封面右下角盖公章 |
| 骑缝 | 整本文件侧面骑缝盖公章 |
| 签字页 | 法定代表人或授权人签字+名章 |
| 变更页 | 每页右下角盖公章（或变更章） |
| 复印件 | 加盖公章 + "与原件一致"章 |

### 签字要求

| 角色 | 签字要求 |
|------|---------|
| 法定代表人 | 必须签字，可不盖章 |
| 授权代理人 | 签字 + 法定代表人授权委托书 |
| 项目负责人 | 视招标文件要求，常见于技术文件 |
| 财务负责人 | 视招标文件要求，常见于报价文件 |

### 鲜章 vs 彩色打印

| 场景 | 要求 |
|------|------|
| 纸质正本 | 必须鲜章（红色原章） |
| 纸质副本 | 可盖鲜章或彩色打印章 |
| 电子投标 | 按招标文件要求，可能需要电子签章 |

## 装订与包装

| 元素 | 要求 |
|------|------|
| 装订方式 | 左侧胶装 或 左侧装订（不用拉链夹/活页夹） |
| 分册装订 | 每册不超过200页，超出时分册 |
| 密封要求 | 正本、副本分开密封 |
| 包封要求 | 外层牛皮纸密封，加盖公章 |
| 份数 | 按招标文件要求，通常1正4副或2正6副 |

## 格式自检脚本

### Python 一键格式修正

```python
"""
format_check.py — 投标文件格式自检与修正
"""
from docx import Document
from docx.shared import Pt, Cm
from docx.enum.text import WD_ALIGN_PARAGRAPH
from docx.oxml.ns import qn


def check_format(docx_path, verbose=True):
    """检查DOCX格式是否符合规范"""
    doc = Document(docx_path)
    issues = []

    for p in doc.paragraphs:
        if not p.text:
            continue
        for run in p.runs:
            # 检查字体
            if run.font.name not in ('宋体', '仿宋', '黑体', '楷体', None):
                issues.append(f"[字体] 非标准字体: {run.font.name}")
            # 检查字号（1pt = 2磅，小四=24磅=12pt）
            if run.font.size and run.font.size.pt not in (10.5, 12, 14, 15, 16, 18):
                issues.append(f"[字号] 非标准字号: {run.font.size.pt}")
        # 检查行距
        if p.paragraph_format.line_spacing not in (1.5, None):
            issues.append(f"[行距] 非标准行距: {p.paragraph_format.line_spacing}")

    # 检查页边距
    for section in doc.sections:
        for margin_name, margin_val in [
            ('上', section.top_margin),
            ('下', section.bottom_margin),
        ]:
            if abs(margin_val.cm - 2.54) > 0.1:
                issues.append(f"[页边距] {margin_name}边距{ margin_val.cm:.2f}cm（应为2.54cm）")

    if verbose:
        if issues:
            print(f"发现 {len(issues)} 个格式问题：")
            for issue in issues:
                print(f"  - {issue}")
        else:
            print("✅ 格式检查通过")

    return issues


def fix_format(docx_path, output_path):
    """批量修正格式：将所有段落修正为宋体12pt 1.5倍行距"""
    doc = Document(docx_path)

    for p in doc.paragraphs:
        pf = p.paragraph_format
        pf.line_spacing = 1.5
        pf.line_spacing_rule = None  # 使用数字行距而非倍数

        for run in p.runs:
            run.font.name = '宋体'
            run._element.rPr.rFonts.set(qn('w:eastAsia'), '宋体')
            if not run.font.size:
                run.font.size = Pt(12)

    doc.save(output_path)
    print(f"✅ 格式修正完成: {output_path}")
```

### 使用方式

```powershell
# 检查格式
python format_check.py 主公司V5.docx

# 批量修正格式
python format_check.py 主公司V5.docx 主公司V5_修正.docx --fix
```

## 各文件类型格式差异

| 文件类型 | 字体 | 字号 | 页边距 | 行距 |
|---------|------|------|--------|------|
| 技术应答文件 | 宋体 | 小四 | 3.17cm | 1.5倍 |
| 商务标书 | 宋体 | 小四 | 2.54cm | 1.5倍 |
| 价格标书 | 宋体/黑体 | 小四/三号 | 2.54cm | 固定值28磅 |
| 资质复印件 | — | — | — | — |
| 图纸（建筑类） | 黑体 | 五号 | 2.54cm | — |

**最重要的原则：以招标文件中的格式要求为准。** 以上所有参数均为无明确要求时的默认参考值。
